Documentation 1 - Modified (<int>)
Comment with code "Modified" following by integer e.g. "(1)" for "Modified (1)" indicate the line has been modified.
List of "Modified (<int>)" status:
- <int>     : For numbering or identifier category of modified.
- Disabled  : Commenting the line and not be used.
- Change    : Commenting above line and changed line at the bottom.
- Addition  : Commenting between additional line.
